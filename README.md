# Laboratório de Visão Computacional

Repositório dedicado a estudos, experimentos e protótipos em Visão Computacional, com foco em detecção de objetos utilizando YOLO, técnicas de rastreamento e aplicações em Edge AI.


## Objetivo

Este repositório tem como finalidade o aprendizado e experimentação das seguintes áreas:

- Fundamentos de Visão Computacional
- Deep Learning aplicado a imagens
- Detecção de Objetos com YOLO (Ultralytics)
- Inferência em tempo real com OpenCV
- Técnicas de rastreamento de objetos
- Conceitos de Edge AI (deploy em Jetson)
- Experimentos para integração com Digital Twin

O objetivo final é evoluir de experimentos básicos até aplicações reais em ambientes industriais, como detecção de pessoas e cargas em sistemas de guindaste.


## Tópicos estudados

- Conceitos de Redes Neurais
- Redes Neurais Convolucionais (CNNs)
- Diferença entre classificação e detecção
- Arquitetura e funcionamento do YOLO
- Preparação e anotação de datasets
- Treinamento e avaliação de modelos
- Métricas de desempenho (mAP, precision, recall)
- Inferência em vídeo em tempo real
- Rastreamento de múltiplos objetos (SORT / DeepSORT / ByteTrack)
- Deploy em dispositivos embarcados (Jetson / TensorRT)


## Tecnologias

- Python
- PyTorch
- Ultralytics YOLO
- OpenCV
- Google Colab
- Jupyter Notebook
- NVIDIA Jetson
